DANCEMUSIC WTF-300 is a quiz game for learning how the different electronic dance music styles sound like.
Created by Mattias Hemmingsson.

The game uses over 500 music tracks from 57 different subgenres in Electronic Dance Music. The dataset is based upon musicmap.info and its public YouTube playlists.

If you find any bugs or have feedback, file an issue on the project's GitHub page or email me.
