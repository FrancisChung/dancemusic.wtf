DanceMusic.WTF was created by Mattias Hemmingsson in the urge to learn what the fuck he's dancing to.

The game contains over 500 individual tracks spread across 57 different dance music genres.

The dataset is based upon musicmap.info’s public playlists. If you want to learn more about music genres and their relations I’d recommend you heading over there.

If you find any bugs or have feedback, file an issue on the project's GitHub page or email me.
